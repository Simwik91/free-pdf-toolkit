# Free PDF Toolkit

This project is a free, browser-based PDF toolkit that allows users to merge, split, compress, and convert PDF files.

## Features

-   **Merge PDFs**: Combine multiple PDF files into one document.
-   **Split PDF**: Extract specific pages or split by page ranges.
-   **Compress PDF**: Reduce file size while maintaining quality.
-   **Convert PDF**: Convert between PDF and image formats (JPG, PNG).
-   **Privacy-Focused**: All processing happens in the browser. No data is sent to any server.

## Usage

1.  Open `index.html` in a web browser.
2.  Select a tool: Merge, Split, Compress, or Convert.
3.  Upload your file(s).
4.  Follow the on-screen instructions to process your file(s).
5.  Download the result.
